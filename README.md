# KIDE - Kirjallisuuden edistämiskeskus

Tämä on yksilöprojekti kurssille. Sivusto on tehty Reactilla ja Vitellä.

## Teknologiat

- React 19
- Vite
- CSS3

## Fontit

- **Body teksti**: Public Sans (Google Fonts)
- **Otsikot**: DM Serif Display (vaihtoehto PP Hattonille)

## Asennus

```bash
npm install
```

## Kehitysympäristö

```bash
npm run dev
```

Avaa selaimessa: `http://localhost:5173`

## Rakentaminen

```bash
npm run build
```

## Projektin rakenne

- `src/components/` - React komponentit
- `public/images/` - Kuvat ja SVG tiedostot
- `src/App.jsx` - Pääkomponentti
- `src/main.jsx` - Entry point
